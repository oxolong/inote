# CNN（卷积神经网络）

## 概述
卷积神经网络是一种专门处理网格状拓扑结构数据的深度学习模型，在计算机视觉领域取得了巨大成功。其核心思想是通过局部连接和权重共享大大减少参数量。

## 核心思想
1. **局部感受野**：每个神经元只连接输入的一小块区域
2. **权重共享**：同一卷积核在整个输入上滑动，共享参数
3. **池化降采样**：逐步减小特征图尺寸，提取高层特征

## 网络结构
```
输入图像 → 卷积层 → ReLU → 池化层 → ... → 全连接层 → 输出
```

## 经典架构
- **LeNet-5**（1998）：手写数字识别，7层
- **AlexNet**（2012）：ImageNet 冠军，引入 ReLU 和 Dropout
- **VGGNet**（2014）：使用小卷积核（3x3），加深网络
- **ResNet**（2015）：引入残差连接，可训练 152 层

## 应用领域
- 图像分类、目标检测、语义分割
- 人脸识别、视频分析
- 医学影像分析
