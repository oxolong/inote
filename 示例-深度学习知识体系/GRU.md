# GRU（门控循环单元）

## 概述
GRU 是 LSTM 的简化变体，合并了遗忘门和输入门为「更新门」，参数更少，训练更快，同时保持了与 LSTM 相当的建模能力。

## 与 LSTM 的对比
| 特性 | LSTM | GRU |
|------|------|-----|
| 门控数量 | 3（遗忘/输入/输出） | 2（重置/更新） |
| 细胞状态 | 有独立的 c_t | 无，h_t 兼作记忆 |
| 参数量 | 较多 | 较少（约 LSTM 的 75%） |
| 速度 | 较慢 | 较快 |

## 门控公式
```
重置门: r_t = σ(W_r · [h_{t-1}, x_t] + b_r)
更新门: z_t = σ(W_z · [h_{t-1}, x_t] + b_z)
候选值: h̃_t = tanh(W_h · [r_t ⊙ h_{t-1}, x_t] + b_h)
隐藏态: h_t = (1 - z_t) ⊙ h_{t-1} + z_t ⊙ h̃_t
```

## 使用建议
- 数据量较小时优先选择 GRU（防止过拟合）
- 数据量充足时 LSTM 和 GRU 效果相当
- 需要双向建模时 Bi-GRU 是常见选择
