# 线性回归

## 概述
线性回归是最基础的回归模型，假设目标值与输入特征之间存在线性关系。

## 数学表达
$$y = w_1x_1 + w_2x_2 + ... + w_nx_n + b$$

其中 $w_i$ 为权重参数，$b$ 为偏置项。

## 损失函数
**均方误差（MSE）**：
$$L = \frac{1}{N}\sum_{i=1}^{N}(y_i - \hat{y}_i)^2$$

## 优化方法
- **最小二乘法**：解析解，$w = (X^TX)^{-1}X^Ty$
- **梯度下降**：迭代优化，适用于大规模数据

## 优点
- 实现简单，可解释性强
- 计算效率高

## 局限性
- 只能学习线性关系
- 对异常值敏感
- 容易欠拟合
