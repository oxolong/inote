# LSTM（长短期记忆网络）

## 概述
LSTM 是 RNN 的改进版本，通过引入门控机制和细胞状态，有效解决了标准 RNN 的梯度消失问题，能够学习长距离依赖。

## 核心组件
- **细胞状态（Cell State）**：贯穿时间步的"记忆高速公路"
- **三个门控**：
  - **遗忘门**：决定丢弃哪些旧信息
  - **输入门**：决定存储哪些新信息
  - **输出门**：决定输出哪些信息

## 门控公式
```
遗忘门: f_t = σ(W_f · [h_{t-1}, x_t] + b_f)
输入门: i_t = σ(W_i · [h_{t-1}, x_t] + b_i)
候选值: c̃_t = tanh(W_c · [h_{t-1}, x_t] + b_c)
更新:   c_t = f_t ⊙ c_{t-1} + i_t ⊙ c̃_t
输出门: o_t = σ(W_o · [h_{t-1}, x_t] + b_o)
隐藏态: h_t = o_t ⊙ tanh(c_t)
```

## 优点
- 有效缓解梯度消失，可以学习数百步的长依赖
- 在多种序列任务上表现优异

## 缺点
- 参数较多（4 组权重矩阵）
- 计算相对昂贵
- 仍难并行化
