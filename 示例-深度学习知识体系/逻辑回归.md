# 逻辑回归

## 概述
逻辑回归是一种用于二分类问题的经典模型。尽管名称中包含"回归"，但它实际上是一个分类算法。

## 核心思想
在线性组合的基础上，通过 Sigmoid 函数将输出映射到 (0, 1) 区间：
$$\hat{y} = \sigma(w^Tx + b) = \frac{1}{1 + e^{-(w^Tx + b)}}$$

## 损失函数
**交叉熵损失（Cross-Entropy Loss）**：
$$L = -\frac{1}{N}\sum_{i=1}^{N}[y_i\log(\hat{y}_i) + (1-y_i)\log(1-\hat{y}_i)]$$

## 决策边界
- 通常以 0.5 为阈值
- $\hat{y} \geq 0.5$ → 正类
- $\hat{y} < 0.5$ → 负类

## 扩展
- **多分类**：通过 Softmax 函数扩展为多项逻辑回归
- **正则化**：L1（Lasso）或 L2（Ridge）正则化防止过拟合
