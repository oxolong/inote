# 池化层

## 概述
池化层（Pooling Layer）对特征图进行降采样，减小特征图尺寸，降低计算量，同时保留重要特征。

## 主要类型

### 最大池化（Max Pooling）
- 取池化窗口内的最大值
- 保留最显著的特征
- 常用配置：2x2 窗口，stride=2

### 平均池化（Average Pooling）
- 取池化窗口内的平均值
- 保留整体信息
- 常用于网络末端的全局平均池化（GAP）

## 作用
1. **降维**：减少特征图空间尺寸
2. **降低计算量**：减少后续层的参数和运算
3. **防止过拟合**：降低特征分辨率，相当于正则化
4. **平移不变性**：对输入的微小偏移不敏感

## 与卷积步长的对比
- 池化：无参数，仅降采样
- 步长卷积：有参数，同时提取特征和降采样（现代趋势）
